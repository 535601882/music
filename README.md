# music 青柠音乐 微信小程序

# 数据来自酷狗音乐
> 只供学习开发使用
> http://m.kugou.com

# 功能特性
+ 获取新歌榜
+ 获取排行榜
  + 获取榜单
+ 获取歌单
  + 获取歌单列表
+ 获取歌手
  + 获取歌手列表
    + 获取歌手信息
+ 播放界面


# 待优化
+ 歌手列表跟歌曲列表还没有做滚动加载
+ 数据还未做本地缓存，导致每次打开页面就要从新从远程重新取数据
+ 播放界面还未完成
